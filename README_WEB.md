# Photocopy Scheduler - Streamlit Web

## Chạy local

```bash
pip install -r requirements_web.txt
streamlit run streamlit_app.py
```

## Icon

App tự tìm icon theo thứ tự:

1. `assets/icons`
2. `utils/icons`
3. `/Users/mac/web_photocopy_task_scheduler/assets/icons`

Nên đặt icon vào `assets/icons` trong project để dễ chạy trên máy khác hoặc deploy.
